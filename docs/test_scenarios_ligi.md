# Тестови сценарии — Етап 5 Лиги

## 1. Създай лига (OK)

```
Създай лига Първа лига 2025/2026
```

Очакван резултат: `Лига 'Първа лига' (сезон 2025/2026) е създадена.`

---

## 2. Създай същата лига втори път (ERROR)

```
Създай лига Първа лига 2025/2026
```

Очакван резултат: `Лига 'Първа лига' за сезон '2025/2026' вече съществува.`

---

## 3. Добави отбор (OK)

```
Добави отбор Левски в лига Първа лига 2025/2026
```

Очакван резултат: `Отбор 'Левски' е добавен в лига 'Първа лига' (2025/2026).`

---

## 4. Добави същия отбор втори път (ERROR)

```
Добави отбор Левски в лига Първа лига 2025/2026
```

Очакван резултат: `Отбор 'Левски' вече е добавен в лига 'Първа лига' (2025/2026).`

---

## 5. Покажи отбори в лига (OK)

```
Покажи отбори в лига Първа лига 2025/2026
```

Очакван резултат: списък с клубове в лигата.

---

## 6. Генерирай програма с 3 отбора (ERROR)

Сценарий: създай лига, добави 3 отбора, опитай генериране.

```
Генерирай програма Тестова лига 2025/2026
```

Очакван резултат: `Недостатъчно отбори за програма (има 3, минимум 4).`

---

## 7. Генерирай програма с 4+ отбора (OK)

Сценарий: преди това добави 4 отбора (Левски, ЦСКА, Лудогорец, Ботев) в лига.

```
Генерирай програма Първа лига 2025/2026
```

Очакван резултат: отчет с брой кръгове, мачове, примерен 1-ви кръг.

- 4 отбора → 3 кръга, 6 мача
- 6 отбора → 5 кръга, 15 мача
- 5 отбора → 5 кръга, 10 мача + BYE

---

## 8. Генерирай повторно (ERROR)

```
Генерирай програма Първа лига 2025/2026
```

Очакван резултат: `Лига '...' вече има генерирана програма. Използвайте 'прегенерирай програма ...'.`

---

## 9. Прегенерирай програма (OK)

```
Прегенерирай програма Първа лига 2025/2026
```

Очакван резултат: старата програма се изтрива, нова се генерира.

---

## 10. Невалиден сезон (ERROR)

```
Създай лига Втора лига 2025-2026
```

Очакван резултат: `Невалиден формат на сезон '2025-2026'. Използвайте YYYY/YYYY (напр. 2025/2026).`

---

## 11. Добави несъществуващ клуб (ERROR)

```
Добави отбор НЕСЪЩЕСТВУВАЩ в лига Първа лига 2025/2026
```

Очакван резултат: `Клуб 'НЕСЪЩЕСТВУВАЩ' не е намерен.`

---

## 12. Покажи програма (OK)

```
Покажи програма Първа лига 2025/2026
```

Очакван резултат: всички кръгове и мачове.

---

## Относно round-robin алгоритъма

Използва се **circle method**:

1. При четен N → N-1 кръга, N/2 мача на кръг
2. При нечетен N → N кръга, (N-1)/2 мача + 1 BYE на кръг
3. Един отбор е фиксиран, останалите се ротират
4. Всяка двойка играе точно веднъж
5. Няма мач A срещу A
