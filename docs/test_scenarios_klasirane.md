# Тестови сценарии — Етап 7 Класиране

## Архитектура

```
handler → service → repo
```

- `handler` — валидира вход
- `service` — изчислява класирането от `matches` (само `played`)
- `repo` — SQL заявки за изиграни мачове

## Правила за точкуване

| Резултат | Точки |
|---|---|
| Победа | 3 |
| Равен | 1 |
| Загуба | 0 |

## Сортиране (Tiebreakers)

1. Точки (`PTS`) — низходящо
2. Голова разлика (`GD`) — низходящо
3. Вкарани голове (`GF`) — низходящо
4. Име на отбор — азбучно

## Източник на данни

Класирането се изчислява **изцяло от таблица `matches`** — само мачове със `status = 'played'`:
- Мачове с `NULL` резултат или `status = 'scheduled'` **НЕ** участват
- Ръчно въвеждане на точки/класиране е **забранено**
- Отбори без мачове се показват с нули

## Chatbot команда

```
Покажи класиране <лига> <сезон>
```

Пример:
```
Покажи класиране Първа лига 2025/2026
```

---

## 1. Лига без изиграни мачове → таблица с нули

Сценарий: създадена лига с 4 отбора, без генерирана програма.

```
Покажи класиране Първа лига 2025/2026
```

Очакван резултат:
```
Класиране: Първа лига (сезон 2025/2026)
  (няма изиграни мачове)
   # Отбор                MP  W  D  L   GF:GA   GD PTS
   1. Ботев                 0  0  0  0  0:0   +0   0
   2. Левски                0  0  0  0  0:0   +0   0
   3. Лудогорец             0  0  0  0  0:0   +0   0
   4. ЦСКА                  0  0  0  0  0:0   +0   0
```

---

## 2. Един мач 2:1 → правилни точки

Сценарий: Левски 2-1 ЦСКА.

```
Покажи класиране Първа лига 2025/2026
```

Очакван резултат:
```
   1. Левски                1  1  0  0  2:1   +1   3
   2. ЦСКА                  1  0  0  1  1:2   -1   0
   ...
```

- Левски: 3 точки (победа), 2 вкарани, 1 допуснат
- ЦСКА: 0 точки (загуба), 1 вкаран, 2 допуснати

---

## 3. Два мача → правилно натрупване

Сценарий: Левски 2-1 ЦСКА, Лудогорец 1-0 Ботев.

Очаквани точки:
- Левски: 3 (1 победа)
- Лудогорец: 3 (1 победа)
- ЦСКА: 0 (1 загуба)
- Ботев: 0 (1 загуба)

При равенство в точки (Левски и Лудогорец): сортиране по GF → Левски (2) пред Лудогорец (1).

---

## 4. Равни точки → правилно сортиране

Сценарий: Левски 2-1 ЦСКА, Лудогорец 1-0 Ботев.

И двата отбора имат по 3 точки. Сортиране:
1. Точки: равно
2. GD: Левски +1, Лудогорец +1 → равно
3. GF: Левски 2, Лудогорец 1 → **Левски напред**

---

## 5. Равенство + нов мач → 4 точки

Сценарий: добавя се Левски 0-0 Ботев.

- Левски: 3 + 1 = 4 точки (W=1, D=1)
- Ботев: 0 + 1 = 1 точка (D=1)

---

## 6. Scheduled мач не се брои

Сценарий: добавя се мач ЦСКА-Лудогорец със status='scheduled' и NULL резултат.

Този мач **НЕ** участва в класирането. MP на ЦСКА и Лудогорец остават непроменени.

---

## 7. Несъществуваща лига → ERROR

```
Покажи класиране Несъществуваща 2020/2021
```

Очакван резултат: `Лига 'Несъществуваща' за сезон '2020/2021' не е намерена.`

---

## 8. Лига без отбори → ERROR

Сценарий: създадена лига без добавени отбори.

Очакван резултат: `Няма отбори в лига '...' (...).`

---

## Формат на изхода

```
Класиране: Първа лига (сезон 2025/2026)
   # Отбор                MP  W  D  L   GF:GA   GD PTS
   1. Левски                2  1  1  0  2:1   +1   4
   2. Лудогорец             1  1  0  0  1:0   +1   3
   3. Ботев                 2  0  1  1  0:1   -1   1
   4. ЦСКА                  1  0  0  1  1:2   -1   0
```

| Колона | Значение |
|---|---|
| # | Позиция |
| Отбор | Име на клуба |
| MP | Изиграни мачове |
| W | Победи |
| D | Равни |
| L | Загуби |
| GF:GA | Вкарани:Допуснати |
| GD | Голова разлика (+/-) |
| PTS | Точки |
