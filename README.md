# Футболен клуб Chatbot

Chatbot за управление на футболни клубове с Python + SQLite.

## Структура

```
src/
  main.py           — чат цикъл
  db.py             — връзка с база данни
  clubs_service.py  — CRUD операции за клубове
  chatbot.py        — разпознаване на интенции
  intents.json      — дефиниции на команди
sql/
  schema.sql        — SQL схема
```

## Как се стартира

```bash
python src/main.py
```

## Примерни команди

```
> Добави клуб Левски София
> Покажи всички клубове
> Изтрий клуб Ботев
> Промени клуб Левски София на Левски
> помощ
> изход
```
