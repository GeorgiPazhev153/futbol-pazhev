# Футболен клуб Chatbot

Chatbot за управление на футболни клубове, играчи и трансфери с Python + SQLite.

## Архитектура

```
src/
├── main.py                 — чат цикъл (NLU → Router → Services → DB)
├── chatbot/
│   ├── intents.json        — дефиниции на команди (regex)
│   ├── nlu.py              — разпознаване на интенции
│   └── router.py           — диспечиране към services
├── services/
│   ├── clubs_service.py    — CRUD клубове
│   ├── players_service.py  — CRUD играчи
│   └── transfers_service.py — трансфери + транзакции
├── database/
│   └── db.py               — SQLite връзка + transaction helper
└── utils/
    └── logger.py           — логване в commands.log
sql/
  schema.sql                — SQL схема (clubs, players, transfers)
```

## Как се стартира

```bash
# Стартиране на чатбота
python src/main.py

# Зареждане на тестови данни (клубове + играчи + трансфери)
python seed_data.py
```

## Примерни команди

```
── Клубове ──
> Добави клуб Левски София
> Покажи всички клубове
> Изтрий клуб Ботев
> Промени клуб Левски София на Левски

── Играчи ──
> Добави играч Стилиян Петров в Левски позиция MF номер 10
> Покажи играчи на Левски
> Смени номер на Стилиян Петров на 8
> Смени статус на Христо Стоичков на контузен
> Изтрий играч Георги Иванов

── Трансфери ──
> Трансфер Стилиян Петров от Левски в Лудогорец 2026-03-10
> Трансфер Мартин Петров от Ботев Пловдив в Левски 2025-08-15 сума 150000
> Покажи трансфери на Иван Петров
> Покажи трансфери на Левски

── Други ──
> помощ
> изход
```
